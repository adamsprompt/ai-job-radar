## 🎯 The Mission
As an AI Implementation Consultant, my goal is to bridge the gap between technical "noise" and business "signals." The AI Job Radar is a custom-built intelligence tool designed to solve the "Search Friction" in the Denver technology market. It uses a proprietary weighted scoring algorithm to identify high-ROI opportunities that match a "Techno-Functional" profile.

## 📈 Quantifying Business ROI
To demonstrate the tangible financial impact of automated intelligence, this system tracks the delta between manual discovery and algorithmic processing.

$$ROI_{\%} = \frac{(Total\ Monthly\ Time\ Saved \times Hourly\ Rate) - Infrastructure\ Costs}{Infrastructure\ Costs}$$

*Where Total Monthly Time Saved is derived from the delta between manual search (approx. 40 hours) and automated processing (approx. 2 hours).*

## 🧠 The Intelligence: 0-100 Scoring Engine
Unlike standard job boards, this app utilizes a multi-layered evaluation logic:
* **Title Match (30 pts):** Weighted towards Implementation, Transformation, and Strategy roles.
* **Skills Density (50 pts):** Deep-scan for high-value keywords (Generative AI, Agentic AI, LangChain, Change Management).
* **Location Intelligence (10 pts):** Hard-coded focus on the Denver Metropolitan Area and high-tier Remote positions.
* **Target Company Bonus (+12 pts):** Strategic "multiplier" for high-growth firms and industry leaders.

## 🛠️ The Implementation Stack
* **Frontend:** React (Vite) + Tailwind CSS + Lucide React (UI/UX)
* **Backend:** Node.js + Express
* **Database:** PostgreSQL (Production-grade data persistence)
* **Data Pipeline:** SerpAPI (Google Jobs) + Python-JobSpy
* **Security:** Replit Secrets + Resend API (Encrypted environment variables & secure email delivery)
